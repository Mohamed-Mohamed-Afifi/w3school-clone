child-properites
align-self
  • Move in the perpendicular direction
  • flex-start : this child will set at the beginning of container in perpendicular direction
  • flex-end : this child will set at the end of container in perpendicular direction
  • center: this child will set at the center of container
order
  • if it equal -1 this child will appear at the beginning of container
  • order of elements appearence depend on the order number ,if element take less number it will appear first
flex-grow
  • 0 : is the defalut value
  • we use flex grow to make element grow to fill white space in container to make elements grow equally set flex-grow 1
flex-shrink
  • 1 : is the default value of element
  • 0 : this make element take constant width if the container width is shrunk
flex-basis
  • flex-basis detect width if the direction is row
  • flex-basis detect height if the direction is column
shorthand
  • flex : [flex-grow] [flex-shrink] [flex-basis]
